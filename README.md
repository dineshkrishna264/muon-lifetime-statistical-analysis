# Muon Lifetime Statistical Analysis

## Overview

This project analyzes muon decay times to estimate the mean lifetime of the muon using statistical methods. The decay process is modeled with an exponential distribution, and the lifetime parameter is estimated using Maximum Likelihood Estimation (MLE).

## Contents

* `muon_lifetime_analysis.ipynb` — Jupyter notebook containing the full analysis
* `Data_Science_Project_Report.pdf` — Detailed report of the methodology and results

## Methods Used

* Maximum Likelihood Estimation (MLE)
* Chi-square and Kolmogorov–Smirnov tests
* Confidence intervals
* Bootstrap resampling
* Model comparison using AIC and BIC

## Results

The estimated muon lifetime is consistent with the expected value. Statistical tests indicate that the exponential model provides a good fit to the data and is strongly preferred over a Gaussian alternative.

## How to Use

Open the notebook:

```
muon_lifetime_analysis.ipynb
```

and run the cells to reproduce the analysis.
